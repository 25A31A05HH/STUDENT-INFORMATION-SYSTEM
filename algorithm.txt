ALGORITHM: STUDENT INFORMATION SYSTEM (MULTIPLE STUDENTS)

1. Start

2. Create a list (vector) to store multiple student objects

3. Display menu:
      1. Add Student
      2. Add Marks
      3. View All Reports
      4. Search Student by ID
      5. Exit

4. Read user choice

5. If choice == 1:
      a. Create new student object
      b. Input student name and ID
      c. Add student to list

6. If choice == 2:
      a. Enter student ID
      b. Search student in list
      c. If found:
            - Input marks for subjects
         Else:
            - Display "Student not found"

7. If choice == 3:
      a. Loop through all students
      b. For each student:
            - Calculate average
            - Assign grade
            - Display report

8. If choice == 4:
      a. Enter student ID
      b. Search student in list
      c. If found:
            - Display report
         Else:
            - Display "Student not found"

9. If choice == 5:
      Exit program

10. Repeat steps 3–9 until user exits

11. End
